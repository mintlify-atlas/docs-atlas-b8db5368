---
title: "Admin competition endpoints"
description: "Full CRUD for competitions within a tournament — requires ADMIN JWT."
---

<Note>
  All endpoints on this page require a valid **ADMIN** JWT passed in the `Authorization: Bearer <token>` header.
</Note>

Base path: `/api/v1/admin/tournaments/{tournamentId}/competitions`

---

## List competitions (admin view)

```
GET /api/v1/admin/tournaments/{tournamentId}/competitions
```

Returns all competitions for a tournament with admin-level detail including inscription counts and revenue totals.

### Path parameters

<ParamField path="tournamentId" type="integer" required>
  The ID of the tournament.
</ParamField>

### Response

Returns an array of `CompetitionResponseDTO` objects.

<ResponseField name="id" type="integer">
  Unique identifier of the competition.
</ResponseField>

<ResponseField name="nombre" type="string">
  Name of the competition.
</ResponseField>

<ResponseField name="descripcion" type="string">
  Description of the competition.
</ResponseField>

<ResponseField name="fecha" type="string">
  Date of the competition.
</ResponseField>

<ResponseField name="cupo" type="integer">
  Maximum number of participants. `null` or `0` means unlimited.
</ResponseField>

<ResponseField name="precioBase" type="number">
  Base registration price.
</ResponseField>

<ResponseField name="estado" type="string">
  Availability status (`CON_CUPO` or `SIN_CUPO`).
</ResponseField>

<ResponseField name="cantidadInscripciones" type="integer">
  Total number of active inscriptions.
</ResponseField>

<ResponseField name="montoTotalInscripciones" type="number">
  Total revenue generated by inscriptions for this competition.
</ResponseField>

### Example

```bash
curl -X GET http://localhost:8080/api/v1/admin/tournaments/1/competitions \
  -H "Authorization: Bearer <token>"
```

```json
[
  {
    "id": 3,
    "nombre": "100m Llanos",
    "descripcion": "Carrera de velocidad",
    "fecha": "2026-04-15",
    "cupo": 20,
    "precioBase": 500.0,
    "estado": "CON_CUPO",
    "cantidadInscripciones": 12,
    "montoTotalInscripciones": 6000.0
  }
]
```

## Create a competition

```
POST /api/v1/admin/tournaments/{tournamentId}/competitions
```

Creates a new competition within the specified tournament.

### Path parameters

<ParamField path="tournamentId" type="integer" required>
  The ID of the tournament to add the competition to.
</ParamField>

### Request body

<ParamField body="name" type="string" required>
  Name of the competition. Cannot be blank.
</ParamField>

<ParamField body="cupo" type="integer" required>
  Maximum number of participants. Must be at least `1`. A value greater than `0` creates a capacity-limited competition (`CompetenciaConCupo`).
</ParamField>

<ParamField body="precio" type="number" required>
  Registration price. Must be `0.0` or greater.
</ParamField>

### Response — `201 Created`

Returns the created `CompetitionResponseDTO` (see fields above in the list endpoint).

### Example

```bash
curl -X POST http://localhost:8080/api/v1/admin/tournaments/1/competitions \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "100m Llanos",
    "cupo": 20,
    "precio": 500.0
  }'
```

### Error responses

| Status | Description |
|--------|-------------|
| `400`  | Validation failed (missing name, cupo < 1, or negative price). |
| `404`  | Tournament not found. |

## Update a competition

```
PUT /api/v1/admin/tournaments/{tournamentId}/competitions/{competitionId}
```

Replaces all mutable fields of a competition.

### Path parameters

<ParamField path="tournamentId" type="integer" required>
  The ID of the tournament.
</ParamField>

<ParamField path="competitionId" type="integer" required>
  The ID of the competition to update.
</ParamField>

### Request body

<ParamField body="name" type="string" required>
  Updated name. Cannot be blank.
</ParamField>

<ParamField body="cupo" type="integer" required>
  Updated capacity. Must be at least `1`.
</ParamField>

<ParamField body="precio" type="number" required>
  Updated price. Must be `0.0` or greater.
</ParamField>

### Response — `200 OK`

Returns the updated `CompetitionResponseDTO`.

### Example

```bash
curl -X PUT http://localhost:8080/api/v1/admin/tournaments/1/competitions/3 \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "100m Llanos (Actualizado)",
    "cupo": 25,
    "precio": 550.0
  }'
```

## Delete a competition

```
DELETE /api/v1/admin/tournaments/{tournamentId}/competitions/{competitionId}
```

Permanently removes a competition from the tournament.

### Path parameters

<ParamField path="tournamentId" type="integer" required>
  The ID of the tournament.
</ParamField>

<ParamField path="competitionId" type="integer" required>
  The ID of the competition to delete.
</ParamField>

### Response — `204 No Content`

No response body is returned on success.

### Example

```bash
curl -X DELETE http://localhost:8080/api/v1/admin/tournaments/1/competitions/3 \
  -H "Authorization: Bearer <token>"
```

### Error responses

| Status | Description |
|--------|-------------|
| `404`  | Tournament or competition not found. |

## List inscriptions for a competition

```
GET /api/v1/admin/tournaments/{tournamentId}/competitions/{competitionId}/inscriptions
```

Returns all participant inscriptions for a specific competition.

### Path parameters

<ParamField path="tournamentId" type="integer" required>
  The ID of the tournament.
</ParamField>

<ParamField path="competitionId" type="integer" required>
  The ID of the competition.
</ParamField>

### Response

Returns an array of `InscriptionResponseDTO` objects.

<ResponseField name="id" type="integer">
  Unique identifier of the inscription.
</ResponseField>

<ResponseField name="participanteId" type="integer">
  ID of the enrolled participant.
</ResponseField>

<ResponseField name="participanteEmail" type="string">
  Email of the enrolled participant.
</ResponseField>

<ResponseField name="participanteNombre" type="string">
  Full name of the enrolled participant.
</ResponseField>

<ResponseField name="precioFinal" type="number">
  Final price paid for the inscription.
</ResponseField>

<ResponseField name="fechaInscripcion" type="string">
  Date of inscription in ISO 8601 format (`YYYY-MM-DD`).
</ResponseField>

### Example

```bash
curl -X GET http://localhost:8080/api/v1/admin/tournaments/1/competitions/3/inscriptions \
  -H "Authorization: Bearer <token>"
```

```json
[
  {
    "id": 42,
    "participanteId": 7,
    "participanteEmail": "juan@example.com",
    "participanteNombre": "Juan Pérez",
    "precioFinal": 500.0,
    "fechaInscripcion": "2026-03-18"
  }
]
```
